# Live Calls for OpenVBX

This plugin allows you to view and modify calls in progress.

## Installation

[Download][1] the plugin and extract to /plugins

[1]: https://github.com/chadsmith/OpenVBX-Plugin-Live-Calls/archives/master

## Usage

Manage Calls from the OpenVBX sidebar.

1. Click Live Calls in the OpenVBX Sidebar
2. Find the call you want to modify
3. Select the Flow to redirect the call to
